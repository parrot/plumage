Plumage is a package manager for the Parrot VM module ecosystem. With it, you can perform tasks such as searching, installing, and testing modules in the ecosystem.

The ecosystem currently includes various compilers targeting Parrot, libraries, language bindings, and more.

Dependencies

Parrot

Download

You can download Plumage in either zip or tar formats.

You can also clone a local copy of the repository with Git by running: 

git clone git://github.com/parrot/plumage.git plumage

Install

To install Plumage, run the following commands: 

parrot setup.pir build
parrot setup.pir test
parrot setup.pir install

For more information on using Plumage, see the wiki.
