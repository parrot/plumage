Plumage is a package manager for the Parrot VM module ecosystem. With it, you can perform tasks such as searching, installing, and testing modules in the ecosystem.

The ecosystem currently includes various compilers targeting Parrot, libraries, language bindings, and more.

Dependencies

Parrot

Download

You can download Plumage in either zip or tar formats.

You can also clone a local copy of the repository with Git by running: 

git clone git://github.com/parrot/plumage.git plumage

Install

To install Plumage, run the following commands: 

parrot setup.pir build
parrot setup.pir test
parrot setup.pir install

For more information on using Plumage, see the wiki.

Usage

Getting started with Plumage is very simple. Suppose you want to try out a new programming language but are not sure which. Let's see what's available: 

plumage projects
The projects command lists all the projects in the ecosystem. Maybe you're feeling kinda silly and want to write some lolcode: 
plumage install lolcode
This builds, tests, and installs a particular project/module; in this case, lolcode. It could be that you're not really in a lolcatz mood right now. Uninstalling a project is done with the aptly named uninstall command: 
plumage uninstall lolcode
There's a lot more you can do with Plumage. You can get more help by running: 
plumage usage
