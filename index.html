plumage by parrot

package manager for Parrot

Plumage is a package manager for Parrot. It can be used to easily install various projects, libraries, and compilers for Parrot

Dependencies

Parrot

Install

parrot-nqp setup.nqp build
parrot-nqp setup.nqp test
parrot-nqp setup.nqp install


For more information on using Plumage, see the wiki.
